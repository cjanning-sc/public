Senior Principal Systems Engineer (Top Secret Clearance Required)
R10016909
1st Shift (United States of America)

This is a senior level position in charge of a systems classified server environment.  The selected candidate will be expected to understand server infrastructure and how it relates to: Storage systems, backup systems, power systems and HVAC systems.  Will need to be able to plan for additional sever capacity based on the direction of the program.  Need to be able to plan projects, deal with suppliers, maintain budgets and assign tasks.  The candidate should have good project management skills with the ability to manage Systems related projects.  Must be able to plan and design new systems based on a given set of requirements and assist the Systems Administrator team when issues arise. 

This position will need to be able to communicate to senior leadership on the state of the Datacenter, its needs and future directions.  A strong knowledge of the Microsoft Server, and/or Linux/Unix operating systems is desirable. Must have working knowledge of some of the following technologies: Storage Area Networks, file server and email server systems, as well as other technologies like backup solutions, and virtualization.

 Basic Qualifications: 

  • A minimum of 9 years of Server Systems with a Bachelors in Science; 7 Years with Masters; 4 Years with PhD
  • Active DoD Top Secret Clearance
  • Ability to obtain and maintain program access
  • 8140/8570 Certification or the ability to pass Security+ before hire date.

Preferred Qualifications: 

  • Active SCI clearance
  • Strong leadership and presentation skills
  • Knowledge of:
  • Microsoft Server
  • RHEL
  • Scripting/PowerShell
  • Email server (Exchange)
  • The ability to communicate with Senior Leadership
  • Microsoft Certification a Plus
  • Microsoft Office proficiency
