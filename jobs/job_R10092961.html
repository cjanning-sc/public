Northrop Grumman is seeking a System Integration Engineer to perform system engineering, integration, system administration and cyber security for submarine and surface training systems for the U. S. Navy at the Naval Undersea Warfare Center in Newport, RI.  You will work with latest generation submarine tactical systems for combat control, weapon control and electronic warfare to support integration into a large-scale high-fidelity training environment.  You will contribute to the design, development implementation and deployment of simulation/stimulation-based training systems.

Job Responsibilities:

The responsibilities for this position are to design, develop, install, and integrate training systems that support both the Naval Submarine and Surface Ship fleets. Qualified applicants should have engineering experience related to design, development, and integration. Qualified applicants will perform the following tasks:

  • Install, configure, test, and maintain operating systems, system services, application software and system management tools for Linux and Windows based systems, both natively and within virtual machines
  • Monitor and test application performance to identify issues with installation and performance and to identify possible solutions and work with developers to implement fixes
  • Troubleshoot complex computer systems and networking to diagnose and suggest/implement potential solutions
  • Develop and support a complex build process organized to ensure correct and repeatable integration and delivery
  • Support physical hardware installation of IT computing equipment and cabling at both local and remote sites

Basic Qualifications:

  • Minimum of bachelor's degree in Computer Science/Engineering or other related field and 5 years’ experience
  • Experience integrating Linux and Windows systems and subsystems.
  • Linux operating systems, preferably Red Hat Enterprise Linux RHEL or Community Enterprise Operating System (CentOS) distributions.
  • Installing, configuring, and troubleshooting
  • Ability to identify operating system issues and produce patches to fix issues, as required.
  • Ability to obtain a Secret Clearance
  • Ability to work on site in Rhode Island.

Preferred Qualifications:

  • Active Security Clearance
  • Experience with Submarine Tactical and Electronic Warfare Systems, including AN/BLQ-10, at the operation level and/or build/deployment level
  • Experience with debugging issues with the Linux kernel including examining source code and analyzing kernel crashes
  • Experience with Gluster, NFS, and other direct attach file systems
  • Experience with System Build Automation Tools (Linux kickstart, Ansible, Puppet)
  • Networking knowledge (OSI Network Layers, TCP/IP, DHCP)
  • Current IAT Level I Certifications
    • Network+
    • Security+
    • Linux+
  • Computer/Electronics Hardware knowledge
  • Configuration management experience with both centralized and distributed version control systems (e.g., SVN and Git)
  • Version Control best practices for using them, as well as familiarity with different software CM development workflows, branching and merging strategies
  • Experience with virtualization technologies, preferably Red Hat KVM (e.g., Red Hat KVM, VMware, libvirt/virsh)
