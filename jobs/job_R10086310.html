Description

The prospective candidate will perform software development and maintenance using Java, C and scripting languages in Linux and Windows environments.  The successful candidate will assist with all or part of the following: requirements analysis, application design, code development, testing, code maintenance, software install and documentation.

Location: 

In support of the GRASP project located in Stafford, VA.  Occasional travel CONUS/OCONUS may be required.  The candidate will work as part of a mixed team of Government and contractor personnel.

Required Qualifications: 

  • Bachelor's + 5 years of exp: Master's + 3 years of exp; PhD + 0 years of exp
  • Bachelor's + 9 years of exp; Master's + 7 years of exp; PhD + 4 years of exp
  • Demonstrated skills may be substituted for a formal degree.
  • Professional programming experience developing or maintaining software in Java, C or scripting languages such as bash, python or Csh.
  • Successful candidates must be well organized, a good communicator, and capable of multi-tasking and possess strong analytical skills.
  • Ability to work in Windows and RHEL environments.
  • TS/SCI clearance and willing to get a CI polygraph
  • Willing to complete Security+ certification within 3 months of hire.

Desired Qualifications:

  • Although not required, experience with the following technologies would be a plus: Java Swing; multi-threaded and distributed Java applications; Java Web application (e.g., Tomcat) servers or Java Applications servers (e.g. JBoss and/or Wildfly are highly desired); relational databases (JDBC, SQL, and MySQL); JEE (Java Enterprise Edition) experience (JDBC, JNI, JNDI, JMS, Servlet/JSP, EJB/JPA); Web service experience (e.g., RESTFul Web services)
  • Web (Brower based) GUI front end development (HTML, JavaScript, CSS, Angular)
  • Configuration Management experience working with software version control tools (Jira’s Bamboo, Stash, GIT, Subversion)
  • Linux and / or Windows system administration experience would be a plus
