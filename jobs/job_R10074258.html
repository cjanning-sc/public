SCA Computer Systems Analyst 3
R10074258
1st Shift (United States of America)

Northrop Grumman Defense Systems sector (NGDS) is seeking an SCA Computer Systems Analyst 3 to join our growing team of diverse professionals at Fort Sill near Lawton, Oklahoma.  The selected candidate will be responsible for Information Assurance (IA) security and vulnerability assessment and remediation of the Fort Sill Mission Training Complex (MTC) Information Systems.

Responsibilities:

  • Sets up, builds, maintains, configures, troubleshoots, and updates Server Operating system and Applications, applies Information Assurance Vulnerability Managements (IAVMS) and Security Technical Implementation Guides (STIGs).

  • Runs the Security Content Automation Protocol (SCAP) tool against, and run vulnerability scans on the:

    • Domain Controllers (DC) servers

    • Hypervisors (HV)

    • Host Based Security System (HBSS) servers (to include download from Defense Information Systems Agency (DISA) and apply manually)

    • Network Management System (NMS) servers

    • Assured Compliance Assessment Solution (ACAS) Security Centers and NESSUS systems (to include downloads from DISA and apply manually) and other similar systems as required

    • Standalone systems which support RMF (example: antivirus laptops, management laptops.)

  • Runs the Assured compliance Assessment Solution (ACAS) and Cisco vulnerability scans on all Risk Management Framework (RMF) IA-enabled network devices/systems.

  • Updates and maintains RMF documentation for Servers.

  • Assists with STIG checklists implementation.

  • When needed, performs Daily, Weekly, and Monthly checks on HBSS, HVs, DCs, ACAS Security Centers, and NMS

  • When needed, performs monthly backups for logs and HBSS, HVs, DCs, ACAS Security Centers, and NMS servers

  • Provides exercise support when needed

  • Troubleshoots Network Devices hardware and software

  • Assists Spoke sites with network and server install and documentation

  • Scans peripherals (internal and external) for viruses

  • Installs weekly DAT files

  • Sets up the network within an organization and maintains the network infrastructure, such as routers, switches, and Firewalls

Basic Qualifications:

  • One of the following:

    • High school diploma or GED with a minimum of 8 years of technical or systems administration experience

    • Bachelor’s degree in a technical discipline with a minimum of 4 years of technical or systems administration experience

    • Master’s degree in a technical discipline with a minimum of 2 years of technical or systems administration experience

  • Must have System Administrative knowledge of Microsoft Server 2016, 2019, Windows 10, Red hat Linux, SQL, HBSS, ACAS, HV and VMware.  Must be able to obtain certifications for all Operating Systems support (this can be done on an approved Army site)

  • Must have knowledge of Network Monitoring Software SolarWinds, Network Policy (RADIUS), Active Directory, and Domain Controller

  • Must hold or obtain Cisco Certified Network Associate (CCNA) certification or obtain on approved Army site training

  • Valid CompTIA Security+ CE certification (Network + and A+ are helpful)

  • Must have an active U.S. Department of Defense (DoD) Secret security clearance

  • Must be able to work on-site at Fort Sill, near Lawton, Oklahoma

  • Must be able to travel as needed (up to 10% of the time)

Preferred Qualifications:

  • CompTIA Network+ and/or A+ certifications

  • Valid CCNA certification

  • Valid Operating Systems support certifications

SCA hourly rate: $34.74; H&W as specified in the applicable wage determination
