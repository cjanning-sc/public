Northrop Grumman Aeronautics Systems has an opening for a Principal Systems Engineer on the System Integration team working in the Software Release Documentation area. This System Integration Engineer will be part of our team of qualified, diverse individuals. This position will be in Melbourne, FL. The selected candidate will create and coordinate the documentation necessary to release software and describe the testing which was performed on multiple programs. This selected engineer will serve as a liaison between software, the system test team, and software configuration management.  

Essential Functions:
• Support software release activities on multiple programs and teams with high levels of engagement and collaboration throughout system engineering.
• Prepares, develops and seeks approval for the Project Coordination Memo (PCM) for the release of SW 
• Prepares, develops and brings to the Technical Review Board (TRB) the System Test Report (Software Release Memo – SRM)  
• Ability to coordinate and communicate with multiple groups, understand artifacts from each group and extract information for each software release document  
• Develops necessary reports and metrics to enable systems engineering decisions
• Responsible for supporting the required documentation and the release of software on schedule for the E-2D program.  
• Candidate shall display professionalism, motivation, leadership skills, good communications skills and the ability to work both independently and with multiple teams. 
• The selected candidate should thrive in a fast-paced work environment with high expectations, significantly diverse assignments, collaborative/team settings across all levels. 

Basic Qualifications:
- Bachelor's degree with 6 years of experience on documentation and presentations for Large Military Aircraft Systems 
- Active Secret Clearance, awarded/reviewed within the last 5 years, and the ability to obtain and maintain Special Program Access

Preferred Qualifications:
- Aircraft Development Experience
- Knowledge of JIRA and DOORS tool
- High level of skill and experience with Microsoft Office Suite
