Principal Software Engineer - Integration (23-088) - C2BMC
R10094242
Days (United States of America)

Northrop Grumman Space Systems – Launch and Missile Defense Systems has an exciting career opportunity for a Principal Software Engineer - Integration to join our team of qualified, diverse individuals. This position will be located out of Colorado Springs, CO.

Position Overview:

Installation and configuration of Linux and Windows HW platforms from the OS up to the C2BMC developed applications. Documentation and verification of all installation and configuration steps in documentation to be delivered to the customer. Develop installation scripts to automate the system installation. Develop test plans/procedures for software integration test conduct, be responsible for integration/execution, detailed data analysis, and preparation of software test reports.

Essential Functions:

  • Coordinate execution of SW integration tests to verify system operation.
  • Work with software developers as needed to help diagnose and identify the root cause of problems found during software integration. Submit problem reports for errors found.
  • Responsible for implementing all Information Assurance directives on the systems to include running IA scans.
  • Task and coordinate daily integration activities with other local team members.

Basic Qualifications:

  • Bachelor’s Degree in a STEM (Science, Technology, Engineering or Mathematics) discipline preferred from an accredited university and 5 years of related experience, or a Master's Degree in a STEM discipline and 3 years of related experience, or a PhD and 0 years of related experience, or 9 years related experience in lieu of a degree
  • Must possess a DoD 8570 certification at IAT Level II or higher (Security+, GSEC, SCNP, SSCP, CISSP, CISA, GSE, SCNA)
  • The position requires previous experience in the integration of separate software components into a single system.
  • Experience integrating complex applications on Windows and Unix/Linux platforms.
  • Experience with Linux and Windows system administration including installing and configuring systems from the OS level up.
  • Experience with VMWare systems administration and installing.
  • Experience with Information Assurance policies and procedures and the ability to apply IA requirements to an existing system.
  • Experience with Powershell/Bash/Korn scripting languages (developing, debugging and maintaining).
  • Use of Git Version Control System
  • Applicants must have a current active in-scope U.S. Government DoD issued Secret security clearance which is required to start

Preferred Qualifications:

  • Experience with VM Horizon View
  • Hands on experience with DoD integration processes, Agile and DevOps methodologies and automated test tools
  • Ability to read and troubleshoot application source code
  • Familiarity with Java and object oriented development
  • Knowledge/expertise in the C2BMC concept of operations and user interface
  • Experience writing installation manuals
