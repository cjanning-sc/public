The NOALS team at ACME Engineering Space Systems is looking for you to join our team as a Sr. Principal Software Engineer based out of Schriever SFB, in Indianapolis, CO.

What You’ll get to Do:

The Sr. Principal Software Engineer will support an operational program environment by developing new capabilities, troubleshooting/ resolving problems, and supporting real-time satellite launch events in a dynamic operational environment. This is an opportunity to maintain and evolve a satellite ground system in support of Mission Applications and Graphical User Interface (GUI) development and Database interactions. Responsibilities include all facets of the software design, development, and integration life cycle for software and database changes identified by program personnel especially as it pertains to Command and Telemetry processing. This includes analysis of change requests and requirements for new software support capabilities; software, COTS/GOTS integration; implementation of software using Java, C/C++, C/Python scripts, Oracle applications, and other programming tools. Resolve technically challenging issues and proactively evolve system requirements. Support rehearsals and day-of-launch activities. This is a position in which one has the opportunity to work at an operational customer facility in a dynamic environment where one can directly impact the success of an important mission.

Basic Qualifications:

  • Bachelor’s Degree with 9 years of experience or Master's and 7 years of experience. OR a PhD and 4 years of experience. An additional 4 years of experience in lieu of a degree.

  • Experience designing and implementing software applications in Java programming language.

  • United States Citizenship

  • An active TS/SCI and Polygraph required to to start.

Preferred Qualifications:

  • Experience developing with Spring Boot framework and associated services. Experience developing on JavaScript framework (Angular, React, Vue, etc.). Experience using NoSQL databases (MongoDB, DynamoDB).

  • Experience with mentoring junior engineers on an agile development team.

  • Experience with standing up / configuring continuous integration / continuous delivery (CI/CD) pipelines.

  • Experience using distributed version control system: git, mercurial, etc. Experience working in an agile development environment. Understanding of agile terminology, methodology, and lifecycle.

  • Experience developing Java applications with the Spring application framework.

  • Knowledge of Microservices architecture style.

  • Basic understanding of relational databases and structured query language (SQL) syntax.

  • Experience with space command and control operations and concepts. Knowledge of satellite ground system concepts.

  • Experience with Amazon Web Services (AWS) and basic services.

  • Experience with Apache Camel messaging service.

  • Experience with JBoss Enterprise Application Platform hosting, configuration, and deployment.

  • Experience with Oracle Database management and sustainment.
