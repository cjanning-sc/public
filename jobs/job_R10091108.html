Sr. Principal Software Engineer IV

The successful candidate for this position will be responsible for the design and implementation of embedded software, simulation, and test tools through the full software life cycle for launch vehicles. This includes the development of software requirement specifications and design documentation as well as participating in software process activities including peer reviews, integration testing, and configuration management. The candidate may also work in conjunction with a software project manager to develop schedules, track progress, lead a development team, and support internal and external program and customer meetings.

A security clearance or the ability to obtain a security clearance is also required. Candidates should have experience with large real-time embedded systems and the full software development lifecycle. Experience with software project management is a plus.

Required Qualifications:

  • Object Oriented (OO) software development experience with the C++ programming language
  • Experience in embedded real-time software development, debug and test
  • Experience with real time operating systems (RTOS) environments including VxWorks, Integrity, LynxOS or other equivalent RTOS
  • Proficient written and verbal communication skills
  • Experience with full software development lifecycle

Desired Qualifications:

  • Experience with PowerPC processors (PPC 750, 8260 & 8280)
  • Interrupt Service Routine, Device Driver, and chip level software development and debugging experience
  • Working knowledge of hardware/software analysis tools (debuggers, logic analyzers, oscilloscopes, emulators, etc)
  • Proficiency with Linux and Linux based software development tools
  • Familiarity with DOORS (Requirement Management System), Rational Rose, Rhapsody, Razor and Synergy tools
  • Experience with CMM/SEI or equivalent software development process
  • Experience with UML in developing design models
  • Experience with space craft or launch systems software development is highly desired.

Minimum Qualifications:

9 Years with BS; 7 Years with Masters; 4 Years with PhD

Role is full time onsite

Must have an active and current secret clearance
