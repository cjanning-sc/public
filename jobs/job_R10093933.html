ACME Engineering Space Systems is seeking a Real-Time Embedded Software Engineer / Principal Real -Time Embedded Software Engineer. This position can be located in Indianapolis, Indiana and will support the Ground Based Strategic Deterrent (GBSD) program. 

What You’ll Get To Do: 

The selected candidate will be responsible for developing software for the Command & Launch (C&L) Software Team.  Using DevOps and Agile methodologies you will be constantly optimizing systems and services for security, automation, and performance/availability, while ensuring solutions developed adhere and align to the baseline systems designs, configuration management, and architecture standards. You will work closely in a DevOps environment with the program personnel to ensure smooth execution of continuous integration and test. The selected candidate for this role will work on software systems primarily in C/C++ development, with a focus on  Real-Time Operating System (RTOS) software design for Single Board Computers (SBC) (e.g., Wind River, Green Hills Software)

This position’s standard work schedule is a 9/80. The 9/80 schedule allows employees who work a nine-hour day Monday through Thursday to take every other Friday off. This role may offer a competitive relocation assistance package.
