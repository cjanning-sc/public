Northrop Grumman is seeking a User Interface Developer. This position will be located at our Defense Systems Sector in Walpole, MA.

The qualified candidate will become part of Northrop Grumman’s 2Is Solutions team: a software development organization within the Global Sustainment and Modernization division that delivers software solutions and analytical models supporting supply chain management, forecasting, logistics, and pricing analytics.

The User Interface Developer will contribute front-end code to a variety of web and mobile applications. The candidate must have a strong interest in user interface/user experience (UI/UX) and front-end development.

Job duties include, but are not limited to:

  • Writing JavaScript and utilizing associated libraries such as React Native and jQuery
  • Adding to and maintaining custom front-end libraries and blocks for rapid instantiation of software instances
  • Writing HTML and CSS as needed
  • Actively participating in UI design and code reviews
  • Collaborating with stakeholders from a variety of functional roles to deliver high quality software products

Basic Qualifications:

A candidate must meet ALL of the below criteria. The candidate must:

  • Have 2 years experience with Bachelor’s degree; 4 years relevant job experience without Bachelor’s degree
  • Be able to obtain a U.S. Government security clearance (U.S. citizenship is a pre-requisite)
  • Be proficient in HTML, CSS, and JavaScript

Preferred Qualifications:

  • Bachelor’s degree in Computer Science, Software Engineering, Digital Media, Design, or similar
  • Experience with iOS mobile app development
  • Experience with React Native and jQuery libraries
  • Familiarity with web accessibility concepts
