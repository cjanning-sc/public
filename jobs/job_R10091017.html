Northrop Grumman, a leading global security company, is looking to hire cleared Security Coordinator in our Redondo Beach location to support Space Systems Division (SSD) Security Organization.

This position will be on a 9/80 work schedule to work in a lobby environment and to provide Security escort support as needed.  Although Security operations are 24/7, the lobby hours of operations are only on first shift.

The selected candidate will perform the following tasks:

- Control ingress/egress to Northrop Grumman Facilities

- Process employees, visitors and contractor personnel using the Enterprise Security Information System -Monitor lobby activity, reporting unusual or emergency situations to the Security Control Center

- Perform security escorts and conduct security monitor activities

- Perform equipment and access badge inventories

- Assist with evacuation drills and actual emergencies as required

Basic Qualifications:

- High school degree/GED and a minimum of two years security experience

- Current, active Top Secret security clearance

- Must be able to obtain and maintain SCI clearance

- Must be able to function in a Windows based computer environment

Preferred Qualifications:

- TS / SCI clearance

- Bachelor's degree

- 3+ years of security experience.
