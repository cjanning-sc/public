We have an opening for either a Software Engineer or a Principal Software Engineer to join our talented, diverse Sagittarius Ground team in Redondo Beach, CA. You will have the opportunity to design, develop, document, test, and debug software written in JAVA, C++ and Python; ensure that the developed software is effective and takes into consideration performance and security implications, and work in a Agile environment. We utilize a wide variety of DevOps technologies including Atlassian suite of tools (Bamboo, Jira, Bitbucket), Fortify, Ansible, and many more.

Sagittarius is a relatively new (2020) program major Class A satellite program. We are building a constellation of satellites to perform a classified mission. This role would be working on the software for the ground station, which provides command and control of the satellites and their payloads and communications with externals. Our software is a mix of reuse and new development. Most of the new development is done in Java with a Linux backend and Windows front end with JavaFX UIs. We are primarily a service oriented architecture built on a message bus. We are a fully Agile/Devops program featuring 8 software scrum teams and 50+ developers. All software is built in an Atlassian based CI/CD pipeline. The major components of the system are command and telemetry, resource allocation, and test/training/simulation. We are a 50/50 split between unclassified and classified development. Hybrid work arrangements may be supported.
