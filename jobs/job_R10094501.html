ACME Engineering is seeking a Data Scientist / Principal Data Scientist for our Operations organization. This position will be located at our Mission Systems Sector in Cleveland, Ohio. This position can be performed as a hybrid model via telecommuting. You will be leading an exciting, cutting-edge data analytics program enabling data analytics at ACME Engineering.  We are seeking a highly motivated individual, with an eagerness to learn, and a passion for data analysis/mining, reporting and programming. The selected candidate should have a willingness to explore new technologies independently and develop them in a team environment. 

Roles and Responsibilities:

  • Support the manufacturing operations team by recommending/developing/re-engineering visibility reports or tools to translate information into graphs and charts
  • Possess analytical and problem-solving skills necessary for quickly developing recommendations based on quantitative and qualitative data from many different types of sources
  • Decompose user requirements into logical functions/components
  • Design, develop, and maintain the front-end web applications of a scalable analytics system
  • Utilize various toolsets for data mining/data analysis
  • Explore/Develop new technologies or approaches to data mining and interpretation
  • Create ad hoc metrics and reports based on unique business needs
  • Work closely with customer to develop and implement data related architectures, designs, and solutions 
  • Drive the collection of new data and the refinement of existing data sources 
  • Have solid understanding of data warehouse development and optimization techniques 
  • Identify patterns and higher-level information from all the data sources through use of tools and application programming
  • Understand the general problem space and is confident in explaining the different insights generated from the data

This requisition may be filled as a Level 2 or a Level 3 based with the qualifications below.

Basic Qualifications for the Level 2 (Data Scientist):

  • Bachelor's degree or higher in a STEM area (Science, Technology, Engineering or Math) and 2 years of relevant experience (0 year with Master’s degree)
  • Experience in data science, information systems, data management or web programming
  • Must be able to obtain a U.S. government security clearance (U.S. citizenship is a pre-requisite)
  • Must be proficient in MS Office – Excel, SharePoint, Teams, Outlook, PowerPoint
  • Must be proficient in data analytics visualization software package - Tableau, Power BI
  • Must be proficient in database technologies - SQL, Oracle

Basic Qualifications for the Level 3 (Principal Data Scientist):

  • Bachelor's degree or higher in a STEM area (Science, Technology, Engineering or Math) and 5 years of relevant experience (3 years with Master’s degree)
  • Experience in data science, information systems, data management or web programming
  • Must be able to obtain a U.S. government security clearance (U.S. citizenship is a pre-requisite)
  • Must be proficient in MS Office – Excel, SharePoint, Teams, Outlook, PowerPoint
  • Must be proficient in data analytics visualization software package - Tableau, Power BI
  • Must be proficient in database technologies - SQL, Oracle

Preferred Qualifications:

  • Experience in or knowledge of a manufacturing environment
  • Experience with object-oriented languages (e.g., JAVA, VBA, Python, SQL, R, C+)
  • Experience with AGILE project management software (e.g., Atlassian Jira/Confluence/Bitbucket, Microsoft Azure DevOps).
  • Experience with process workflow software (e.g., SharePoint)
  • Experience with Robotic Process Automation software (e.g., Blue Prism)
  • Experience with Web Design/Development
  • Experience with data transportation formats (e.g., JSON, XML)
  • Experience using a version control system for code management (e.g., Git, SVN).
  • Experience with modern web development stack (e.g., ASP.NET, MVC, JS Frameworks, CSS, HTML)
