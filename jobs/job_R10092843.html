Principal Cyber/Software Security Assurance Engineer - Secret Clearance(Space/Satellite Systems)– SECRET in Dulles
R10092843
Days (United States of America)

This is a DevSEC Ops position in a 100% Agile development environment focused on ground segment and mission application software security; it is not a software development position.  Cyber Software Security Assurance Engineers are focused on software security assurance activities, such as Static and Dynamic security scanning of application software; advocating for Secure Coding Best Practices, other application security testing; and environment/infrastructure hardening per CM Standards (STIGs); security infrastructure configuration and optimization for tools such as Splunk, RSA Tokens, OpenSEC, SSC SCAP Tool, ACAS/Nessus, HBSS/ENS/McAfee suite.  

Additional Responsibilities for a Principal Cyber/Software Security Assurance Engineer include: 

  • Working as an independent, technical contributor in an Agile engineering environment, where the Cyber Engineer may assist identifying and prioritizing security requirements; triage of Static Code Analysis (SCA) tool findings (e.g. Fortify) and assist in prioritizing the findings as technical debt in the SwDLC backlog. 

  • Advising software development scrum teams on secure coding practices, security-focused engineering trade studies, and other security best practices. 

  • Conducting system vulnerability scanning, remediation and patch management activities on Windows and Red Hat operating systems and various COTS/GOTS applications, including those within virtualized and/or cloud environments.  

  • Adjudicating security tool findings by working with software developers, systems engineers, and other engineering disciplines to resolve technical and programmatic cybersecurity concerns.  A demonstrated ability to work collaboratively and productively with others is an essential skill. 

  • Documenting Standard Operating Procedures (SOPs), and when needed, performing software patch installation, other flaw remediation, antivirus updates, and continuous monitoring (ConMon) activities. 

  • Ensuring systems are operated, maintained, and disposed of in accordance with security policies and procedures as outlined in the system security authorization package (SSP, SCTM, ConMon Plan, RAR). 

  • Running SCAP SCANs, developing and running NIST 800-53a Verification Procedures and running through manual STIG check lists. 

  • Preparing RMF SSP and related artifacts for delivery to customer and assessors. 

  

Basic Qualifications: 

  • Bachelors Degree in STEM Science, Technology, Engineering, Math field 
  • 5 years IT security experience in any field 
  • Must have U.S. Citizenship and current or active DOD SECRET clearance 
  • Any DoD 8570 IT Security certification 
  • Familiarity with Systems Engineering processes and milestones; understanding of the requirements analysis, decomposition, and allocation process
  • Must be committed to working on-site at least three days per week

 
 Preferred Qualifications: 

  • 3 years experience in the Defense Aerospace Industry 
  • CISSP Certification 
  • Active TS security clearance, with SCI eligibility 
  • Experience working in restricted facilities and hardening highly sensitive IT systems 
  • Demonstrated knowledge and experience participating in Systems Engineering milestone reviews, as the cybersecurity focal
