Northrop Grumman Mission Systems (NGMS) is seeking Principal Embedded Software Engineers and Senior Principal Embedded Software Engineers with the desire to learn new technologies to join our innovative Software Technologies Organization to help develop, enhance and maintain software on cutting edge products and systems. Candidates will work on software systems for mission systems in all phases of the software development lifecycle, primarily in C/C++. 

This position is located in Linthicum/Baltimore, MD.

Summary of Role:

NGMS is seeking Principal and Senior Principal Embedded Software Engineers with experience in state estimation filter techniques, including but not limited to Kalman Filters, for use in processing and filtering data collected by sensor systems and the desire to learn new technologies to join our innovative Software Technologies Organization to help develop, enhance, and maintain software on cutting edge products and systems. You will work on software systems for mission systems in all phases of the software development lifecycle, primarily in C/C++, on an advanced airborne radar.  Any experience with sensor systems and programming sensor system data collection techniques is a plus.

Roles and Responsibilities:

  • You will be implementing detailed math algorithms from models to process signal information as well as interface with systems engineering to develop flow down software requirements and technical specifications

  • Identifying risks, suggesting mitigation, tracking defects, developing resolutions, and reporting status

  • Developing and executing unit tests and build verification test plans, and track and report test status

  • Conducting reviews of software related products, software designs, and additional technical documents

  • Participating in Integrated Product Teams (IPT), design reviews, and other reviews, as required

  • Participating in the technical peer reviews to ensure system/software requirements are being met from an end-to-end system point of view
