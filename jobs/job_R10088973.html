Program Overview:

The Data Exploitation organization is looking for either a Principal Software/Systems Engineer (level 3) or a Senior Principal Software/Systems Engineer (level 4) to work on our team which involves various parts of the ground Overhead Persistent Infrared (OPIR) processing system.  General tasks consist of but not limited to: software application development, updating existing applications with new features, containerizing new and existing applications, deploying containers using Kubernetes, fixing issues reported by end user, creating automated tests to test applications, updating/creating algorithms, and analysis of system performance. 

The candidate will have the opportunity to design, document, and test, software to run on a customer specified target framework. The candidate will ensure that the developed software is effective and takes into consideration performance and security implications. As stated above, we will employ agile development and employ DevSecOps technologies.

Basic Qualifications for Principal Software Engineer :

  • Bachelor's degree in STEM with 5+ years of relevant experience, or a Master's degree in STEM with 3+ years of relevant experience for Principal Software Engineer
  • Experience with C++ and OOP concepts
  • Experience with Atlassian products
  • Current active DoD Secret Clearance or higher

Basic Qualifications for Senior Principal Software Engineer :

  • Bachelor's degree in STEM with 9+ years of relevant experience, or a Master's degree in STEM with 7+ years of relevant experience for Sr Principal Software Engineer
  • Experience with C++ and OOP concepts
  • Experience with Atlassian products
  • Current active DoD Secret Clearance or higher

Preferred Qualifications:

  • Excellent communication and organizational skills
  • Strong interpersonal and analytic skills
  • Thrive on colleague interactions and collaborations
  • Proficiency in software configuration management and a version control tools ( Git/GitHub, Bitbucket, some knowledge on ClearCase)
  • Knowledge of Agile development concepts, methodologies, and associated tools (Jira/Atlassian tools)
  • Some experience with software design, architecture, and code
  • Some experience developing and/or implementing complex algorithms
  • Some experience with Static Code Analysis (SCA) tools (Fortify, Understand, SonarQube)
  • Some experience with unit testing (tools, frameworks, automation)
  • Some experience with containerization (Docker, Kubernetes, OpenShift)
  • Linux operating system administration experience (administration would be a plus)
  • Active and current DoD Top Secret clearance.
