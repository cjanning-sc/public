ACME Engineering is seeking a Linux System Administrator to perform system administration, network administration and cyber security supporting submarine training systems development for the Naval Undersea Warfare Center in Newport, RI.  You will work with latest generation Intel based servers. Linux cluster configurations and virtualization architectures using both VMWare vSphere and Redhat KVM.  You will support developers supporting software development toolsets including source control management, software development environments, and overall Linux System Administration.  You will contribute to the design, development implementation and deployment of simulation/stimulation-based training systems infrastructure.

Job Responsibilities:

The responsibilities for this position are to install, and integrate software development infrastructure that is used to produce software and systems to for both the Naval Submarine and Surface Ship fleets.

Responsibilities:

  • Design computer systems and networks to support development infrastructure and high-fidelity submarine trainers
  • Install and maintain operating systems
  • Design and implement efficient backup solutions for system infrastructure
  • Monitor and test application performance to identify issues with installation, performance, and identify possible solutions while working with developers to implement fixes
  • Develop and support build process
  • Troubleshoot computer systems and networking to diagnose and suggest/implement potential solutions
  • Support hardware and software installation of IT computing equipment and cabling of system infrastructure
