Banks Allowed to Delay Implementation of E-Payment User Protection Guidelines to 30 June 2019
C50F7576-68F5-4C05-A736-B5D41E1997B0
MAS will give banks and credit card issuers more time to adopt the E-Payments User Protection Guidelines. The banks have requested MAS for more time to implement the Guidelines, owing to the scale and complexity of system changes needed to implement the transaction notification standards for all products and customers as set out in the Guidelines.
